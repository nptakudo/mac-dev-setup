# mac-dev-setup
For a new mac developing env

Prerequisites:
- Homebrew
- Zsh
- Oh-My-Zsh
- Zsh Plugins
- Zsh Theme
- Zsh Syntax Highlighting
- Zsh Autosuggestions
- Zsh Zsh-autocomplete
- Zsh Zsh-syntax-highlighting

```shell
brew install uv
```

Modify the `~/.zshrc` file according to the source code.
```shell
cat ~/.zshrc
```
